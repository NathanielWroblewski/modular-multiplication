Running locally
---

On a mac, clone the repo, and run a server:

```
$ git clone https://github.com/NathanielWroblewski/modular-arithmetic.git
$ cd modular-arithmetic
$ open http://localhost:8000 && python -m SimpleHTTPServer
```
